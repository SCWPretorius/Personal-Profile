## Overview

This repository serves as a knowledge base with agents for task planning, instructions for coding standards, and skills for common operations like git commits.

## Tech Stack

- **Languages/Frameworks**: TypeScript 5/ES2022, Vue.js 3, Python, Go, PowerShell, SQL, Git, Tailwind CSS v4 + Vite

## How It Works

1. Reference agents (e.g., task-planner.agent.md) for specialized workflows.
2. Apply instructions (e.g., python.instructions.md) for consistent coding.
3. Use skills (e.g., git-commit/SKILL.md) for reusable functionalities.
4. Maintain standards by updating and contributing to the .github/ directory.

## Key Features

- Autonomous agents for development tasks.
- Detailed guidelines for multiple technologies.
- Reusable skills with examples.
- Centralized documentation for workflow consistency.
