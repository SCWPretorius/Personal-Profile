## Overview

Typist is a browser-based typing speed test with a clean, distraction-free interface. Built entirely client-side — no backend, no accounts.

## Tech Stack

- **Frontend**: Vue 3 + TypeScript
- **Persistence**: localStorage for stats history
- **Styling**: Tailwind CSS

## Features

- WPM and accuracy tracking with live per-keystroke feedback
- Custom text import — paste any content to practice with
- Stats history with personal bests and trend charts
- Dark mode (because of course)
- Zero dependencies beyond Vue itself

## Why I Built It

Most typing test sites are cluttered with ads and unnecessary features. Typist is the version I wanted to use — just the test, your stats, and nothing else.
