## Overview

OpenTrack is an open-source habit and goal tracker built with Vue 3, Pinia, and Supabase. It provides a clean dashboard, streak counters, and weekly progress reports to keep you accountable.

## Tech Stack

- **Frontend**: Vue 3 with Composition API and TypeScript
- **State Management**: Pinia
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Styling**: Tailwind CSS

## Key Features

- Habit streaks with visual calendar heatmap
- Weekly summary reports with trend analysis
- Goal milestone tracking
- Real-time sync across devices via Supabase subscriptions
