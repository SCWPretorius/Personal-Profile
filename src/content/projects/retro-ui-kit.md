## Overview

Retro UI Kit is a Tailwind CSS component library designed around a monochrome terminal aesthetic — the same design language powering this portfolio.

## Tech Stack

- **Styling**: Tailwind CSS with custom design tokens
- **Distribution**: npm package
- **Documentation**: Storybook

## Components

- Monospace cards with scanline overlays
- Retro badges and pill tags
- Terminal-style code blocks
- Animated glitch text headings
- CRT-style border treatments

## Design Principles

The kit is built around three constraints: mostly black and white, monospace typography, and deliberate pops of colour for high-impact elements. These constraints produce a distinctive aesthetic that avoids generic AI-generated design.
