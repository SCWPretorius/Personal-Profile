# Hello World

Every developer blog starts with a hello world post. So here's mine.

I've been meaning to write publicly for years. I had the usual excuses — *not enough time*, *nothing interesting to say*, *who would read it anyway*.

## What I'll Write About

Mostly technical stuff:

- Things I learned solving a hard problem
- Framework opinions
- Open-source tools I find useful

## The Stack

This site is built with **Vue 3**, **Vite**, and **Tailwind CSS**. The blog is powered by plain Markdown files — no CMS, no database. Just files. It's fast, portable, and doesn't require a subscription.
